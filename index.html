What is this?

This page demonstrates plenoptic photo rendering using WebGL. Plentopic cameras capture far more information about a scene, which enables effects such as computational refocusing and changing viewpoints.

Go ahead and try changing the sliders above and see how the image changes. If you're curious, you can also see the raw input image.

For more information, see the Wikipedia page.
